{{< callout type="info" icon="graduation-cap" >}}
**Resumen:** Del **4 al 8 de agosto de 2025** participé en **Code/Astro 2025** en **Northwestern University**, un taller orientado a **buenas prácticas de ingeniería de software** para aplicaciones astrofísicas.  
{{< /callout >}}

## Application

Decidí postularme porque quería **consolidar mi flujo de trabajo** para desarrollar **código científico mantenible y abierto**, siguiendo estándares de la comunidad.  
Mi objetivo era llevar estos aprendizajes a la **pipeline de reducción de datos** que estoy construyendo para mi proyecto de maestría.

{{< callout type="success" icon="check-circle" >}}
**¿Qué buscaba lograr?**
- {{< icon name="code-branch" pack="fas" >}} Integrar **Git/GitHub** de forma profesional (ramas, PRs, revisión).
- {{< icon name="book" pack="fas" >}} Crear **documentación clara** y automatizada.
- {{< icon name="flask" pack="fas" >}} Mejorar **pruebas** (unitarias/integración) y **CI**.
- {{< icon name="balance-scale" pack="fas" >}} Adoptar principios de **equidad e inclusión** en la colaboración técnica.
{{< /callout >}}

---

## Workshop

El taller combinó teoría por la mañana y práctica por la tarde. Asistí a **todas** las sesiones opcionales para sacarle el máximo provecho.

**Mañanas — fundamentos y buenas prácticas**
- {{< icon name="laptop-code" pack="fas" >}} Entornos de desarrollo y **paradigmas de programación**.  
- {{< icon name="code-branch" pack="fas" >}} **Git workflows** e integración con **GitHub**.  
- {{< icon name="bug" pack="fas" >}} **Depuración** y **perfilamiento**.  
- {{< icon name="book-open" pack="fas" >}} **Documentación** y **publicación** de paquetes.  
- {{< icon name="vial" pack="fas" >}} **Testing** (pytest) y prácticas de **software inclusivo**.

**Tardes — práctica aplicada**
- {{< icon name="square-root-alt" pack="fas" >}} **NumPy**  
- {{< icon name="table" pack="fas" >}} **Pandas**  
- {{< icon name="chart-line" pack="fas" >}} **Matplotlib**  
- {{< icon name="project-diagram" pack="fas" >}} **Programación paralela**  
- {{< icon name="star" pack="fas" >}} **Astropy** y **Astroplan**  
- {{< icon name="tachometer-alt" pack="fas" >}} **Herramientas de profiling**

{{< callout type="warning" icon="boxes" >}}
Durante la semana trabajé con dos colegas en un **proyecto corto**: paquetizamos, documentamos y publicamos una librería en Python.  
En la sesión final, otros grupos **instalaron y ejecutaron** nuestro paquete sin problemas.
{{< /callout >}}
