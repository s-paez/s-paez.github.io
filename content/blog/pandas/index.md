From August 4 to 8, I participated in the Code/Astro 2025 workshop at Northwestern University. This academic event focuses on developing software engineering skills for astrophysical applications. During the workshop, I engaged in hands-on software development activities and a brief group project. Specifically, the workshop consisted of morning classes that covered development environments, programming paradigms, Git workflows, GitHub incorporation, debugging, documentation, code publishing, testing, and anti-discrimination practices in software development. In the afternoons, I attended all of the optional sessions offered by the workshop, which covered Numpy basics, Pandas, Matplotlib, parallel programming, Astropy, Astroplan, and profiling tools. Throughout the week, I collaborated with two other workshop attendees on a short project involving packaging, documenting, and publishing code. We presented and shared this work during the last session with the other groups, who were able to install it on their own computers.

Attending Code/Astro allowed me to learn the process of writing, preparing, and publishing maintainable, well-documented open source code that follows community best practices. I will now apply what I learned to the data reduction pipeline that I am developing for my master's project.
